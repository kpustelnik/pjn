This text release of the April 2016 version of Walenty consists of the 
following files:

• adjs/walenty_20160418_adjs_verified.txt – adjectival entries verified 
by “superlexicographers”,
• adjs/walenty_20160418_adjs_all.txt – all adjectival entries, including 
those not yet so verified,

• advs/walenty_20160418_advs_verified.txt – adverbial entries verified 
by “superlexicographers”
• advs/walenty_20160418_advs_all.txt – all adverbial entries, including 
those not yet so verified,

• nouns/walenty_20160418_nouns_verified.txt – noun entries verified 
by “superlexicographers”,
• nouns/walenty_20160418_nouns_all.txt – all noun entries, including 
those not yet so verified,

• verbs/walenty_20160418_verbs_verified.txt – verbal entries verified 
by “superlexicographers”,
• verbs/walenty_20160418_verbs_all.txt – all verbal entries, including 
those not yet so verified,

• phrase_types_expand_20160418.txt – possible phrase types extensions.

This and other versions of Walenty, as well as any accompanying
documentation, are available from http://zil.ipipan.waw.pl/Walenty.
